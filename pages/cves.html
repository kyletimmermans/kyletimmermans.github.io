The Blackboard Security Team has been notified of this issue.<\/em><\/p>\n

Update: Blackboard has announced an end-of-life for the People Tool<\/div>\n
\n

Bad actors have the ability to inject arbitrary web script via the Tile<\/code> widget (aka profile-tiles) input forms located in their People Tool<\/code> profile. The input is not properly sanitized and will be stored on the their profile.<\/p>\n
\n

An arbitrary script may be executed on the user\'s web browser (CWE-79).<\/p>\n
\n

Version 9.1 Q2 2017 Cumulative Update 5 (Build: 3200.0.5-rel.6+3dd6b56) and earlier versions that include the Tile<\/code> widget in the profile editor. More than likely will work on later versions with this widget installed, as this issue has not been adressed prior.<\/p>\n
